# react-native-boilerplate

React Native Boilerplate with a basic Authentication Flow built for a quick start using:

- Expo 36.0.0 (RN 0.61.4)
- React Navigation 5
- Redux
- Redux Persist
- Redux Thunk
- ESLint Standard

## Quick start

Install dependencies
```bash
yarn
```

Run application
```bash
yarn start
```
